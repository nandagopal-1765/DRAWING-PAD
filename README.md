# Drawing Pad 🎨

A simple **Drawing Pad Application** built with Python’s Tkinter library.  
Users can draw freely on the canvas using mouse movements.

---

## ✨ Features
- Interactive **Canvas widget** for drawing
- Supports **mouse events** (`<Button-1>`, `<B1-Motion>`)
- Lightweight and beginner-friendly
- Expandable for advanced features (colors, eraser, save image)

---

## 🛠️ Concepts Demonstrated
- **Canvas widget** for graphics
- **Mouse events** for interactivity
- **Event-driven programming** in Tkinter

---

## 🚀 Getting Started
### Prerequisites
- Python 3.x installed
- Tkinter (comes pre-installed with Python)

### Run the Program
```bash
python drawing_pad.py
